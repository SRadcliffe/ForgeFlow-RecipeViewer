# ForgeFlow Recipe Viewer

ForgeFlow Recipe Viewer is a lightweight desktop utility for viewing and inspecting ForgeFlow recipe JSON files generated by ForgeFlow Save Nodes for ComfyUI.

The application allows artists and developers to load recipe exports, inspect generation settings, review prompts, view negative prompts, and recreate previous image or video generations from stored metadata.

## Download

### Windows

Download the latest release:

➡️ **https://github.com/SRadcliffe/ForgeFlow-RecipeViewer/releases/latest**

Available downloads:

* **ForgeFlow Recipe Viewer Setup.exe** (Recommended)
* **ForgeFlow Recipe Viewer.exe** (Portable)

No Node.js or npm installation required.

## Features

* Drag and drop recipe loading
* Image recipe support
* Video recipe support
* Positive prompt viewing
* Negative prompt viewing
* Prompt tab switching
* Copy prompt functionality
* Recipe metadata inspection
* ForgeFlow branded interface

---

## Designed For

Works alongside:

* Forge Save Image
* Forge Save Video
* ForgeFlow JSON Recipe Exports
* ComfyUI Production Workflows

Recipe JSON files generated during image and video exports can be loaded directly into ForgeFlow Recipe Viewer for inspection and reference.

---

## Supported Recipe Data

* File Name
* Seed
* Steps
* CFG
* Sampler
* Scheduler
* Denoise
* Model
* CLIP
* VAE
* Additional custom recipe metadata

## Development

```bash
cd ForgeFlow-RecipeViewer
npm install
npm start
```
