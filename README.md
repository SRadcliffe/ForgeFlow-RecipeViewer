# ForgeFlow Recipe Viewer

ForgeFlow Recipe Viewer is a lightweight Electron desktop application for viewing and inspecting recipe JSON files generated by ForgeFlow Save Nodes for ComfyUI.

The application allows artists and developers to quickly load recipe exports, review generation settings, inspect prompts, view negative prompts, and recreate previous image or video generations from stored metadata.

![ForgeFlow Recipe Viewer](assets/forgeflow-recipe-viewer.png)

## Designed For

ForgeFlow Recipe Viewer is intended to work alongside:

* Forge Save Image
* Forge Save Video
* ForgeFlow JSON Recipe Exports
* ComfyUI Production Workflows

Recipe JSON files generated during image and video exports can be loaded directly into ForgeFlow Recipe Viewer for inspection and reference.

## Features

* Drag and drop recipe loading
* ComfyUI recipe JSON support
* Image recipe support
* Video recipe support
* Positive prompt viewing
* Negative prompt viewing
* Prompt tab switching
* Copy prompt functionality
* Recipe metadata inspection
* ForgeFlow branded interface


## Supported Recipe Data

* File Name
* Seed
* Steps
* CFG
* Sampler
* Scheduler
* Denoise
* Model
* CLIP
* VAE
* Additional custom recipe metadata
